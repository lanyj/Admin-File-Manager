Admin Manager
简单轻便的文件系统

这是一个简单的云端文件存储系统，你可以在这建立自己的新圈子。然后可以与好友一起聊天。文件互传功能正在建设中...与百度网盘相比，改存储系统更加轻量化，操作更加方便，而且可以通过发信息给自己来方便的存储一些重要的文字记录。

消息系统

    负责站内消息互传，输入自己想要联系的用户的用户名，然后输入内容就可以发送。为了保证消息不具有否认性，暂时不提供删除与编辑功能

文件系统

    负责自己的文件管理，将自己的文件上传上去，然后便可以在其他设备上下载使用。

如果你喜欢本网站，可以在Github上找到我。你也可以分享此网站，让更多人加入我们。(:
